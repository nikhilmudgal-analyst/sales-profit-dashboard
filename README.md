# Sales & Profit Dashboard

## Project Overview
This project analyzes sales and profit performance using the Superstore dataset.
The interactive Tableau dashboard provides insights into sales trends, regional
performance, category-wise profit, and sub-category sales.

## Tools Used
- Tableau
- GitHub

## Key Metrics
- Total Sales
- Total Profit
- Total Orders
- Sales Trend by Year
- Profit by State and Region
